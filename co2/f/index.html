PVOS CO2 Instrument REV_F Documentation




PVOS CO2 REV_F firmware / hardware repository on github:

-- Hardware folder, which includes schematic, board layout, KiCad files.

-- Firmware folder, which includes Arduino code. Note that it requires uploading a 'data' folder using an ESP32 upload tool, as well as setting up the Arduino IDE properly with libraries and ESP32 Board support for the Heltec Wifi 32 LoRa v2.
